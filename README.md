# persistent-ui-editor
Excited to share my latest project — **Persistent UI Editor** Chrome Extension!

I developed a browser extension that allows users to:

Edit website UI directly from the browser
Persist changes even after page refresh
Modify text, styles, layout, and elements live
Save customized UI locally in the browser
Instantly preview changes without backend updates

This extension is especially useful for:

* UI testing
* Rapid prototyping
* Client demos
* Temporary UI customization
* Frontend debugging

Tech Stack:

* JavaScript
* Chrome Extension Manifest V3
* Local Storage API
* DOM Manipulation

One of the most interesting parts was implementing persistent DOM changes that remain visible even after refreshing the page.
